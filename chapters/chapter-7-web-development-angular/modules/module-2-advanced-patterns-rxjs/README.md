# Chapter 7, Module 2: Advanced Angular Patterns and RxJS

## Module Overview

This module covers advanced Angular patterns and deep dives into RxJS - the reactive programming library that powers Angular. You'll master observables, operators, and build sophisticated reactive applications.

## Learning Objectives

By completing this module, you will be able to:
- Master RxJS observables and operators
- Implement advanced Angular patterns
- Use Subject for communication
- Handle complex async operations
- Implement change detection strategies
- Create custom directives
- Build reusable component libraries
- Understand Angular's compilation
- Optimize Angular applications
- Implement advanced form patterns

## Curriculum Outline

### Part 1: RxJS Deep Dive
- Observable creation and subscription
- Hot vs cold observables
- Operators overview
- Commonly used operators (map, filter, switchMap, etc.)
- Subject types (Subject, BehaviorSubject, ReplaySubject)
- Higher-order observables and flattening
- Error handling with operators
- Testing observables
- Duration: 3-3.5 hours
- Prerequisites: Chapter 7, Module 1

### Part 2: Advanced Component Patterns
- Content projection (ng-content)
- Component inheritance
- Smart and dumb components
- OnPush change detection
- ViewChild and ContentChild
- Component communication patterns
- Input/Output best practices
- Lifecycle hook optimization
- Duration: 2.5-3 hours
- Prerequisites: Part 1

### Part 3: Directives and Custom Elements
- Structural directives (creating custom)
- Attribute directives
- Directive lifecycle
- ElementRef and Renderer2
- Creating reusable behavior
- Web Components and Angular
- Custom form controls (ControlValueAccessor)
- Directive composition
- Duration: 2.5-3 hours
- Prerequisites: Part 1, Part 2

### Part 4: Performance and Testing
- Change detection optimization
- OnPush strategy mastery
- Zone.js and change detection
- Angular's compilation modes (JIT vs AOT)
- Lazy loading modules
- Bundle analysis and optimization
- Unit testing with Jasmine
- Integration testing
- E2E testing with Cypress/Protractor
- Duration: 2.5-3 hours
- Prerequisites: Part 1, Part 2
