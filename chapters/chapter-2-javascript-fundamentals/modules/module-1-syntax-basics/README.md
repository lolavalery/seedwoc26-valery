# Module 1: Syntax & Basics

## 📚 Module Overview

Learn the fundamental syntax of JavaScript. Master variables, data types, operators, and control flow that form the foundation of all JavaScript code.

## 🎯 Learning Objectives

By completing this module, you will:
- Understand JavaScript syntax
- Work with all data types effectively
- Use operators correctly
- Control program flow
- Understand scope
- Write clean, readable code
- Avoid common JavaScript pitfalls
- Debug basic JavaScript

## 📖 Module Outline

### Part 1: Variables and Data Types
Storing and understanding data in JavaScript

### Part 2: Operators & Control Flow
Making decisions and performing calculations

### Part 3: Loops & Iteration
Repeating operations efficiently

## ⏱️ Duration

- **Estimated time**: 2 weeks
- **Difficulty Level**: ⭐ Beginner

## 📝 Assignments

This module contains hands-on exercises covering:
1. Variable declaration (let, const, var)
2. All data types and type checking
3. Operators and expressions
4. Conditionals (if, switch)
5. Loops (for, while, for-of)
6. Real-world problems

## ✅ Prerequisites

- Chapter 1: Version Control (recommended)
- Text editor (VS Code)
- Modern web browser
- Basic computer knowledge

## 🔗 Next Steps

After completing this module:
- Learn DOM manipulation in Module 2
- Progress to Chapter 3 for advanced concepts
- Build interactive websites

## 💡 Tips

- **Use console.log()**: Debug everything by printing values
- **Test in browser console**: F12 to open developer tools
- **Type examples**: Don't copy-paste - typing builds understanding
- **Experiment**: Change code and see what happens
