## 💾 How to Use the Example

### Option 1: As Reference
```bash
cd project-example
git log --oneline        # Show all 10 commits
git show HEAD            # Show latest commit
# Students view, learn, understand
```

### Option 2: As GitHub Reference
```bash
# Push to your GitHub
cd project-example
git remote add origin https://github.com/YOUR-USERNAME/git-practice-seedwoc.git
git push -u origin main

# Students can see it at github.com/YOUR-USERNAME/git-practice-seedwoc
```

### Option 3: As Starting Template
```bash
# Clone for students to modify
git clone project-example my-project
cd my-project
# Students modify and make their own commits
```
