## 📋 Steps to Push to GitHub

### Step 1: Create GitHub Repository

1. Go to [github.com/new](https://github.com/new)
2. Name: `git-practice-seedwoc`
3. Description: "My first GitHub project from seedwoc Week 1"
4. Choose **Public** (so it's visible)
5. **Skip** "Initialize with README" (we already have one)
6. Click **Create repository**

### Step 2: Connect Local to GitHub

After creating the repo, GitHub will show you commands. Run these in the `project-example` folder:

```bash
# Navigate to the project
cd /c/Users/THE\ EYE\ INFORMATIQUE/OneDrive/Desktop/seed/seedwoc/git_and_github/project-example

# Add GitHub as remote
git remote add origin https://github.com/YOUR-USERNAME/git-practice-seedwoc.git

# Rename branch to main (GitHub default)
git branch -M main

# Push all commits to GitHub
git push -u origin main
```

**Replace `YOUR-USERNAME` with your actual GitHub username!**

### Step 3: Verify on GitHub

1. Go to github.com
2. Click your profile → Repositories
3. Click `git-practice-seedwoc`
4. You should see:
   - ✅ All files visible
   - ✅ All 10 commits in commit history
   - ✅ Complete README displayed
   - ✅ Nice project page

## 💡 Tips for Teaching

1. **Show the commit log**: `git log --oneline --graph`
2. **Explain each commit**: Show what changed in each one
3. **Highlight the branch**: Point out commits 6-7 on feature branch
4. **Clear messages**: Each commit has a clear, descriptive message
5. **Progressive complexity**: Commits build from simple to complex
